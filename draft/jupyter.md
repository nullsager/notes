#python 

Jupyter Notebook 的快捷键是提高效率的关键。很多新手只用鼠标点点点，而熟练工几乎全程不用鼠标。

Jupyter 有一个核心概念叫**“模式（Mode）”**，这也是使用快捷键的前提：
*   **命令模式 (Command Mode)**：左侧显示**蓝色**竖条。按 `Esc` 进入。此时你的键盘指令作用于**整个单元格**（如删除、复制、移动）。
*   **编辑模式 (Edit Mode)**：左侧显示**绿色**竖条。按 `Enter` 进入。此时你的键盘指令作用于**代码文本**（如打字）。

以下是按使用频率排名的最实用快捷键：

### 一、王者级快捷键（每天按几百次）

这几个不记住，Jupyter 基本没法用：

1.  **`Shift + Enter`**：**运行当前代码，并选中下一格**。
    *   *这是最常用的运行方式，因为它让你顺畅地往下写。*
2.  **`Ctrl + Enter`**：**运行当前代码，光标留在原地**。
    *   *场景：你在调试一段代码或调整绘图参数，需要反复运行同一个格子看结果。*
3.  **`Tab`**：**代码补全**。
    *   *场景：输 `pan` 后按 Tab，自动补全为 `pandas`；或者 `pd.read_` 后按 Tab，显示所有读取函数。*
4.  **`Shift + Tab`**：**查看函数文档（神器）**。
    *   *场景：当你写 `pd.read_csv()` 记不住里面参数时，把光标放在括号里，按一下显示简介，按两下显示详细文档。**强烈推荐！***

---

### 二、单元格管理（命令模式，按 Esc 后使用）

确保左侧是**蓝色**条时使用：

1.  **`b`**：在**下方**插入新单元格 (Below)。
2.  **`a`**：在**上方**插入新单元格 (Above)。
3.  **`d, d`** (连按两次 d)：**删除**当前单元格 (Delete)。
4.  **`m`**：将当前格转为 **Markdown** (文本/笔记) 模式。
5.  **`y`**：将当前格转为 **Code** (代码) 模式。
    *   *记忆法：Python 的 y*。
6.  **`z`**：**撤销**对单元格的操作（比如误删了格子，按 z 找回）。
7.  **`l`** (小写 L)：显示/隐藏当前格的**行号**。
    *   *报错时很有用，因为它会告诉你第几行出错了。*

---

### 三、批量操作与整理（进阶）

1.  **`Shift + M`**：**合并**选中的单元格 (Merge)。
    *   *场景：把散落在几个格子的代码合并成一段完整的逻辑。需按住 Shift 选中多个格子后操作。*
2.  **`Ctrl + Shift + -`** (减号)：**拆分**单元格。
    *   *注意：这是在**编辑模式（绿色）**下用的。光标在哪，就从哪里把一个格子切成两个。*
3.  **`x` / `c` / `v`**：剪切 / 复制 / 粘贴单元格。
    *   *不需要选中文字，直接对整个盒子操作。*

---

### 四、代码编辑辅助（编辑模式，绿色）

1.  **`Ctrl + /`**：**注释/取消注释**整行或选中的代码块。
2.  **`Ctrl + [`** 和 **`Ctrl + ]`**：**代码缩进/反缩进**。
    *   *比按 Tab 和 Shift+Tab 更快，尤其是在调整 `if` 或 `for` 循环层级时。*
3.  **`Ctrl + Shift + P`**：**打开命令面板**。
    *   *如果你忘了某个功能的快捷键，按这个，输入你想干的事（比如 "restart"），它会帮你执行。*

---

### 总结：新手必背速查表

建议你把这一小节截图保存，初期对着按：

| 快捷键               | 作用               | 模式     |
| :---------------- | :--------------- | :----- |
| **Shift + Enter** | 运行并跳到下一行         | 通用     |
| **Shift + Tab**   | 查看函数说明书 (光标在括号内) | 编辑 (绿) |
| **Esc**           | 进入命令模式           | 绿 -> 蓝 |
| **Enter**         | 进入编辑模式           | 蓝 -> 绿 |
| **b**             | 在下方加一行           | 命令 (蓝) |
| **d, d**          | 删除当前行            | 命令 (蓝) |
| **m**             | 变笔记模式            | 命令 (蓝) |
| **y**             | 变代码模式            | 命令 (蓝) |

**小贴士**：在 Jupyter 界面中，按 **`h`** (在命令模式下) 可以直接弹出一张完整的快捷键列表速查表。

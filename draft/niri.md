#linux 

```
super + shift + /   展示重要快捷键
super + shift + E   退出niri
super + Q             
```


| 快捷键                           | 功能                   |
| ----------------------------- | -------------------- |
| `super+shift+/ `              | 展示重要快捷键              |
| `super+shift+E`               | 退出 niri              |
| `super+Q`                     | 关闭聚焦的窗口              |
| `super+left` 或 `super+H`      | 将焦点移动到左侧的列           |
| `super+right` 或 `super+L`     | 将焦点移动到右侧的列（上下聚焦同理）   |
| `super+ctrl+left`             | 将列移动到左边              |
| `super+ctrl+right`            | 将列移动到右边              |
| `super+page down` 或 `super+U` | 切换到下方工作区             |
| `super+page up` 或 `super+I`   | 切换到上方工作区             |
| `super+ctrl+page down/U`      | 将列移动到下方工作区           |
| `super+ctrl+page up/I`        | 将列移动到上方工作区           |
| `super+R`                     | 在预设的一组列宽之间循环切换当前列的宽度 |
| `super+F`                     | 最大化当前列               |
| `super+[`                     | 将左侧那一列的窗口吸到当前所在列中    |
| `super+]`                     | 将右侧那一列的窗口吸到当前所在列中    |
| `super+V`                     | 将窗口在浮动和平铺之间切换        |
| `super+shift+V`               | 将键盘焦点在平铺层和浮动层之间切换    |
| `super+O`                     | 打开全局预览图              |
| `ptrSc`                       | 截图                   |
| `super+T`                     | 打开 alacritty 终端      |
| `super+D`                     | 使用 fuzzel 打开应用       |
| `super+alt+L`                 | 锁屏                   |
| `super+shift+HJKL/上下左右`       | 切换显示器焦点              |
|                               |                      |

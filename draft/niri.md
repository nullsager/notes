这是一份针对 **niri**（一个基于滚动布局的 Wayland 合成器）快捷键的详细中文指南。

### Niri 设计理念简介
在阅读快捷键之前，了解 niri 的逻辑很有帮助：
*   **列 (Column)**：niri 的核心单位。一个列可以包含一个或多个窗口（垂直堆叠）。
*   **滚动 (Scrolling)**：所有的列在水平方向上无限排布，像一条长纸带。
*   **吞并/驱逐 (Consume/Expel)**：这是 niri 管理窗口的核心动作，用于将窗口合并进某一列或从列中拆分出来。

### Niri 快捷键功能对照表

| 快捷键                                    | 功能翻译         | 详细讲解                                |
| :------------------------------------- | :----------- | :---------------------------------- |
| **基础操作**                               |              |                                     |
| `Mod + Shift + /`                      | 显示快捷键列表      | 弹出内置的帮助界面，忘记键位时最有用。                 |
| `Mod + T`                              | 打开终端         | 默认调用 Alacritty。                     |
| `Mod + D`                              | 打开应用启动器      | 默认调用 Fuzzel。                        |
| `Super + Alt + L`                      | 锁定屏幕         | 调用 swaylock 锁定系统。                   |
| `Mod + Q`                              | 关闭当前窗口       | 正常关闭当前获取焦点的窗口。                      |
| `Mod + Shift + E` 或 `Ctrl + Alt + Del` | 退出 niri      | 注销并返回登录管理器。                         |
| **焦点移动 (Focus)**                       |              |                                     |
| `Mod + H` / `←`                        | 焦点左移         | 切换到左侧的一**列**。                       |
| `Mod + L` / `→`                        | 焦点右移         | 切换到右侧的一**列**。                       |
| `Mod + J` / `↓`                        | 焦点下移         | 在当前列中，切换到下方的**窗口**。                 |
| `Mod + K` / `↑`                        | 焦点上移         | 在当前列中，切换到上方的**窗口**。                 |
| `Mod + Shift + HJKL/方向键`               | 切换显示器焦点      | 当连接了多个显示器时，将鼠标/焦点移至相邻显示器。           |
| **位置移动 (Move)**                        |              |                                     |
| `Mod + Ctrl + H` / `←`                 | 向左移动列        | 将当前整列向左侧滑动。                         |
| `Mod + Ctrl + L` / `→`                 | 向右移动列        | 将当前整列向右侧滑动。                         |
| `Mod + Ctrl + J` / `↓`                 | 向下移动窗口       | 在当前列内部，将窗口向下移动。                     |
| `Mod + Ctrl + K` / `↑`                 | 向上移动窗口       | 在当前列内部，将窗口向上移动。                     |
| `Mod + Ctrl + Shift + HJKL`            | 将列移至其他显示器    | 将当前整列移动到相邻的显示器。                     |
| **工作区管理 (Workspace)**                  |              |                                     |
| `Mod + U` / `PageDown`                 | 下一个工作区       | 切换到下方的工作区。                          |
| `Mod + I` / `PageUp`                   | 上一个工作区       | 切换到上方的工作区。                          |
| `Mod + Ctrl + U` / `PageDown`          | 移动列至下个工作区    | 将当前选中的列移动到下一个工作区。                   |
| `Mod + Ctrl + I` / `PageUp`            | 移动列至上个工作区    | 将当前选中的列移动到上一个工作区。                   |
| `Mod + Shift + U/I`                    | 移动工作区顺序      | 调整工作区在垂直堆栈中的相对位置。                   |
| **布局管理 (Layout/Tiling)**               |              |                                     |
| `Mod + ,`                              | 吞并 (右侧)      | 将右侧相邻的列“吸入”当前列，变成垂直堆叠状态。            |
| `Mod + .`                              | 驱逐 (底部)      | 将当前列最底部的窗口拆分出去，在其右侧形成新的一列。          |
| `Mod + [`                              | 左侧吞并/驱逐      | 尝试将左侧窗口吸入或将当前窗口向左拆分（取决于上下文）。        |
| `Mod + ]`                              | 右侧吞并/驱逐      | 尝试将右侧窗口吸入或将当前窗口向右拆分（取决于上下文）。        |
| `Mod + R`                              | 循环预设列宽       | 在配置好的几种宽度（如 1/3, 1/2, 2/3）间快速切换。    |
| `Mod + Shift + R`                      | 循环预设行高       | 在配置好的几种窗口高度间快速切换。                   |
| `Mod + F`                              | 最大化列         | 使当前列填满屏幕宽度（再次按下取消）。                 |
| `Mod + C`                              | 居中显示         | 将当前列在视口中间对齐。                        |
| `Mod + V`                              | 切换悬浮/平铺      | 在标准的平铺模式和自由移动的悬浮模式间切换。              |
| `Mod + Shift + V`                      | 焦点切换 (平铺/悬浮) | 快速在悬浮窗口层和平铺窗口层之间切换焦点。               |
| **尺寸调整 (Resizing)**                    |              |                                     |
| `Mod + -` / `Mod + =`                  | 调整列宽         | 减少或增加当前列的宽度（每次 10%）。                |
| `Mod + Shift + -` / `+`                | 调整窗口高度       | 减少或增加当前窗口的高度（每次 10%）。               |
| `Mod + Ctrl + R`                       | 重置高度         | 将窗口高度恢复为自动分配的状态。                    |
| `Mod + Shift + F`                      | 全屏模式         | 窗口进入全屏状态（隐藏状态栏和边框）。                 |
| **截图 (Screenshot)**                    |              |                                     |
| `PrintScreen`                          | 区域截图         | 鼠标选择区域，按 **Space** 保存，按 **Esc** 取消。 |
| `Alt + PrintScreen`                    | 窗口截图         | 截取当前焦点窗口，自动保存并复制到剪贴板。               |
| `Ctrl + PrintScreen`                   | 全屏截图         | 截取整个显示器内容，自动保存并复制到剪贴板。              |

### 给新手的操作建议：
1.  **关于 Mod 键**：默认通常是键盘上的 `Super` 键（Windows 徽标键）。
2.  **核心逻辑**：如果你想把两个窗口并排，它们必须在不同的 **Column** 里；如果你想让它们上下排列，就使用 `Mod + ,` (Consume) 把它们合并。
3.  **配置**：这些快捷键通常在 `~/.config/niri/config.kdl` 文件中定义，你可以根据喜好修改。

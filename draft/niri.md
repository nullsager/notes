#linux 

```
super + shift + /   展示重要快捷键
super + shift + E   退出niri
super + Q             
```


| 快捷键                    | 功能                   |
| ---------------------- | -------------------- |
| `super+shift+/ `       | 展示重要快捷键              |
| `super+shift+E`        | 退出 niri              |
| `super+Q`              | 关闭聚焦的窗口              |
| `super+left`           | 将焦点移动到左侧的列           |
| `super+right`          | 将焦点移动到右侧的列           |
| `super+ctrl+left`      | 将列移动到左边              |
| `super+ctrl+right`     | 将列移动到右边              |
| `super+page down`      | 切换到下方工作区             |
| `super+page up`        | 切换到上方工作区             |
| `super+ctrl+page down` | 将列移动到下方工作区           |
| `super+ctrl+page up`   | 将列移动到上方工作区           |
| `super+R`              | 在预设的一组列宽之间循环切换当前列的宽度 |
| `super+F`              | 最大化当前列               |
| `super+[`              | 将左侧那一列的窗口吸到当前所在列中    |
| `super+]`              | 将右侧那一列的窗口吸到当前所在列中    |
| `super+V`              | 将窗口在浮动和平铺之间切换        |
| `super+shift+V`        |                      |

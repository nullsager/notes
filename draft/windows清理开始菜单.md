Windows 的开始菜单实际上是由两个特殊的文件夹组成的：一个是你个人用户的，另一个是所有用户的。卸载后残留的快捷方式通常就在这两个文件夹里。
步骤：

按 Win + R 键打开“运行”对话框。
检查个人用户的开始菜单文件夹：
输入 shell: programs 然后按回车。
这会打开一个文件资源管理器窗口，路径类似于 C:\Users\你的用户名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs。
在这个文件夹以及它的子文件夹（比如应用开发者创建的文件夹）里，查找你想要删除的应用的快捷方式或空文件夹，然后像删除普通文件一样直接删除它。
检查所有用户的开始菜单文件夹：
再次按 Win + R 键打开“运行”对话框。
输入 shell: common programs 然后按回车。
这会打开另一个文件资源管理器窗口，路径类似于 C:\ProgramData\Microsoft\Windows\Start Menu\Programs。
同样，在这里查找并删除残留的快捷方式或文件夹。
完成这两步后，再次打开开始菜单，那个残留的应用图标应该就消失了。绝大多数情况下，这个方法就能解决问题。

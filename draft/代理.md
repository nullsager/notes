#linux 

将你的机场/订阅链接转换成 Clash 格式的 YAML 文件，并重命名为 `config.yaml` 放入 `/etc/mihomo/ `

编辑 `/etc/mihomo/config. yaml`，确保包含以下内容，以便 Web 面板能连接：

```
port: 7890               # HTTP 代理端口
socks-port: 7891         # SOCKS 5 代理端口
allow-lan: true          # 允许局域网连接
mode: rule               # 模式 Rule/Global/Direct
log-level: info
ipv 6: true

# 外部控制配置 (Web 面板通过这个接口控制)
external-controller: 0.0.0.0:9090
secret: "123456"         # 访问 Web 面板时的密钥，建议修改
```

启动 Mihomo 服务

```
# 重载配置
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start mihomo

# 设置开机自启
sudo systemctl enable mihomo

# 检查运行状态
sudo systemctl status mihomo
```

#cpp 

meeting 类型为 `std::vector<std::vector<int>>`

从小到大排序：

```cpp
std::ranges::sort(meetings, {}, [](auto& a) { return a[2]; });
```

从大到小：

```cpp
std::ranges::sort(meetings, std::ranges::greater{}, [](auto& a) { return a[2]; });
```

传统写法：

```cpp
std::sort(meetings.begin(), meetings.end(), [](const auto& a, const auto& b) {
    return a[2] < b[2]; 
});
```

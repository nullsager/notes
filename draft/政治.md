## 错误表述

+ 货币是资本的唯一物质形态
+ 先抓物质富足，再抓精神文明（应该两手抓）
+ 孙中山发动了北伐战争
+ 劳动力自身的价值是转移到新产品上去的（实际上是再生产出来的）
+ 劳动力商品的市场价格与供需关系无关
+ 内卷会导致企业利润虚高
+ 日本文化渗透目的是镇压中国人民反抗
+ 理想和信念不是思想认识问题，而是实践问题
+ 如果一切从本本出发就不能前进是毛提出来的（其实是邓）
+ 新文化运动是资产阶级新文化开始打破封建文化独占文化阵地局面的起点（其实是维新运动）
+ 垄断会导致整个社会商品的价值不再由生产它们的社会必要劳动时间决定
+ 各国人口因素和自然地理环境的不同是造成社会主义发展道路多样性的决定因素
+ 毛找到了自我革命的方法（其实是习）
+ 甲午战争使中国陷入边疆危机
+ 十一届三中全会称为正本清源、拨乱反正和改革开发的思想指导（关于真理标准问题的大讨论）
+ 科技革命已成为推动社会历史进步的根本动力（其实是人民群众）
+ 在对外贸易上，国家贸易是主体，一定范围内的自由贸易为补充
+ 新事物依然保留旧事物中无法适应新条件的因素
+ 生产力和生产关系共同组成生产的社会形式（其实生产力是生产的物质内容）

## 正确表述

+ 任何事物都可以通过某一中介或一系列中介发生联系
+ 毛提出两个中间地带，三个战略划分
+ 邓提出和平发展两大主题，四项原则
+ 国家资本所有制体现了总资本家剥削雇佣劳动者的关系
+ 实践在认识活动中起决定性作用
+ 孙中山：民族革命是扫除现在的恶劣政治，而社会革命是扫除恶劣政治的根本
+ 毛提出马克思主义同中国第二次结合
+ 习和中亚五国签订“永久睦邻友好合作条约”
+ 虚拟实践是实践活动派生形式，同时又对现实的实践活动产生重大影响
+ 不变资本：不改变自己的价值量，不发生增殖
+ 垄断必须通过竞争来维持
+ 客体能否满足主体需要，是由人的主观愿望决定的
+ 外资经济是社会主义市场经济的组成部分，不是社会主义经济的组成部分

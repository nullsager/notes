`std::setw(int n)`:

+ 设置输出宽度为 n 个字符
+ 只影响下一个输出项
+ 默认右对齐
+ 如果输出内容长度超过设定宽度，则完整输出

```cpp
#include <iostream>
#include <iomanip>

int main() {
    std::cout << std::setw(5) << 42 << std::endl;  // "   42" (左边3空格)
    std::cout << std::setw(5) << "Hello" << std::endl;  // "Hello"
    return 0;
}
```

`std::left`:

+ 设置左对齐
+ 需要与 std::setw 配合使用
+ 一直有效，直到被其他对齐方式覆盖

```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 左对齐，宽度为10
    std::cout << std::left << std::setw(10) << "Hello" << "World" << std::endl;
    // 输出: "Hello     World" (Hello后有5个空格)
    
    // 制作表格样式输出
    std::cout << std::left 
              << std::setw(10) << "Name" 
              << std::setw(5) << "Age" << std::endl;
    std::cout << std::setw(10) << "John" 
              << std::setw(5) << "25" << std::endl;
    return 0;
}
```

常见用法组合：


```cpp
#include <iostream>
#include <iomanip>

int main() {
    // 设置填充字符
    std::cout << std::left << std::setfill('-');
    
    // 创建表格式输出
    std::cout << std::setw(20) << "Product" 
              << std::setw(10) << "Price" << std::endl;
              
    std::cout << std::setw(20) << "Apple" 
              << std::setw(10) << 1.99 << std::endl;
              
    std::cout << std::setw(20) << "Orange" 
              << std::setw(10) << 2.49 << std::endl;
    
    return 0;
}
```

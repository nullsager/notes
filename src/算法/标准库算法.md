```cpp
string s("Hello World");
transform(s.begin(), s.end(), s.begin(), 
    [](unsigned char c) { return toupper(c); });
```

transform第一个参数和第二个参数指定容器的操作返回，第三个参数指定将转换的结果写到哪里，最后的lambda表达式指定如何操作。

可以等价转换为：

```cpp
ranges::transform(s, s.begin(), [](unsigned char c) {
    return toupper(c);
  });
```

这种写法更加简洁

```cpp
word.erase(std::remove_if(word.begin(), word.end(), 
            [](char c) { return ispunct(c); }), word.end());
```

删除word字符串标点符号后面的内容

更简单的写法：

```cpp
erase_if(word, [](const char c) {
      return ispunct(c);  // 判断是否为标点符号
    });
```
